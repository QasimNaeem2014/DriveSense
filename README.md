# Drive Sense

An AI-powered car recommendation and comparison application built with **Streamlit** and **Google Gemini AI**. Drive Sense helps users discover the right vehicle based on their preferences and compare two cars with AI-generated insights.

---

## Features

- AI-powered car recommendations
- Side-by-side car comparison
- Modern dark purple user interface
- Responsive Streamlit design
- Pakistan-focused vehicle recommendations
- Google Gemini AI integration

## Installation

### Clone the repository

```bash
git clone https://github.com/your-username/drive-sense.git
cd drive-sense
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Configure Environment Variables

Create a `.env` file in the project folder.

```env
GEMINI_API_KEY=YOUR_API_KEY
```

### Run the application

```bash
streamlit run app.py
```
